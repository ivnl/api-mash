ABOUT

This website uses Google Api to produce a map and then we can interact with the map to see the current weather using yahoo's weather api, and we also see some important information about that location using the geonames api. Credit goes to Lokesh Dakar for his lightbox creation and the creators + maintainers of all relevant API docs. All icons are from font awesome set.

APIs Used:

Yahoo Weather

Google Maps

Geonames
